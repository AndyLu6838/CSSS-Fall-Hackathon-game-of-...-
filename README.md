# Game of ...

### Getting started:

#### Linux / Mac

`git clone https://github.com/ryan-lintott/CSSS-Fall-Hackathon-game-of-...-.git`

then go to direcvtory containing the file and run

`gcc GameOfLife.cpp -lstdc++` to compile and

`/a.out` to start the game
